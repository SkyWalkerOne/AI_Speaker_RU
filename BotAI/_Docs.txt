--- Как подключить? ---

1) Переместите папку BotAI в папку src в вашем проекте Java (Не обязательно Android);
2) Над классами напишите " import BotAI.Bot; ";
3) Создайте у себя в коде нового бота " Bot (имя бота - любое) = new Bot("Имя, которое бот будет использовать в разговоре", "язык бота"); ";

Пример:
Bot slave = new Bot("Бро", "rus");
*Список доступных языков приведен в конце документа

--- Как пользоваться? ---

Пользоваться ботом можно при помощи одного метода
slave.getAnswer("Ваше сообщение боту");
Данный метод возвращает строку (String), которую можно вывести
Если язык был указан неверно, тогда метод вернет ошибку

Пример:
System.out.println(slave.getAnswer("Привет, как тебя зовут?"));
